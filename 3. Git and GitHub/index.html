A Coder's Simple Cheatsheet to HTML & Git



When diving into the world of web development, understanding the fundamental building blocks of HTML is essential. Whether it's structuring content, embedding media, creating interactive forms, or organizing data, HTML tags and their attributes are the tools that makes the base of a web page. Given below is a concise yet comprehensive guide to some of the most important HTML elements and their key attributes laid out in an easy-to-understand table.







Tag / Element Description Key Attributes / Examples Decription
<h1> to <h6> Headings from largest (<h1>) to smallest (<h6>) <h1>Main Title</h1> Text inside a heading tag will be bold and bigger by default
<p> Paragraph of text <p>This is a paragraph.</p> Defines a paragraph on a web page.
<img> Embeds an image Attributes:
  1. src

  2. alt

  3. height

  4. width

<img src="pic.jpg" alt="Nature" height="200" width="300">
  1. src has the address of the image
  2. alt is the visual aid provided in case the content was not able to load
  3. height defines the height of the image
  4. width defines the width of the image
★ Prefer defining either only height or only width of the element to prevent the image from distortion.
<video> Embeds a video file Attributes:
  1. src

  2. alt

  3. height

  4. width

  5. controls

  6. mute

  7. autoplay

  8. loop

<video src="movie.mp4" controls autoplay></video>
  1. controls will add controls to the video like play, pause, etc.
  2. autoplaywill make the video start on its own when the page loads.
  3. mute mutes the video by default
  4. loop restarts the video on its own when it ends.
★ autoplay only works when the video is muted by default.
<audio> Adds audio player Attributes:
  1. src

  2. alt

  3. height

  4. width

  5. controls

  6. mute

  7. autoplay

  8. loop

<audio src="song.mp3" controls></audio>
<a> Clickable link Attributes:
  1. href

  2. target="_blank"
<a href="https://example.com" target="_blank">Visit Site</a>
  1. href adds the web address to the link.
  2. target="_blank" opens the link in a new tab when clicked.
<ol> Ordered (numbered) list Attribute:
  1. type

  2. start
<ol type="A" start="5">
  1. each item in Ordered List has a unique bullet.
  2. type attribute can change the type of list.
    • type="1" will make the list be in 1, 2, 3...
    • type="a" will make the list be in a, b, c...
    • type="A" will make the list be in A, B, C...
    • type="i" will make the list be in i, ii, iii...
    • type="I" will make the list be in I, II, III...
  3. start defines where the count is going to start from.
<ul> Unordered (bulleted) list <ul>...</ul> unordered list has the same type of bullet throughout the list.
<li> List item (inside ol/ul) <li>Item</li> contains the content of one list item.
<table> Table layout border
<table border="1"> 		Defines the beginning and end of a table.
<tr> Table row <tr>...</tr> Marks the row of the table.
<th> Table header cell <th>Header</th>
  • marks the heading of each column.
  • it is placed in the first row of the table
<td> Table data cell <td>Data</td> describes the data in each cell of the table






Tag / Element Description Key Attributes / Examples Decription
<form> Collects user input <form>...</form> marks the beginning of a form
<input> Editable fields (text, email, etc.) Attributes:
  1. type

  2. placeholder

  3. value

  4. required

  5. readonly

  6. minlength

  7. maxlength

  8. name

<input type="text" name="username" placeholder="Enter Name" required>
<label> Describes input <label for="email">Email:</label> Headings to an input field
<br> Line break <br>
<select> Dropdown menu <select name="country">...</select>
<option> Option in dropdown <option value="India">India</option>
<input type="submit"> Submit button <input type="submit" value="Send">
<input type="radio"> Radio button (single choice) <input type="radio" name="gender" value="male"> Male ★ Add the same name attribute to all radio inputs to let only one option be selected.
<input type="date"> Date picker <input type="date">
<input type="datetime-local"> Date and time picker <input type="datetime-local">
<input type="password"> Password field <input type="password"> Shows masked password






Git Commands Simplified

In the fast-paced world of software development, managing code efficiently and collaborating seamlessly are essential skills. Git, the most popular version control system, empowers developers to track changes, experiment safely, and work together without losing code history. Whether you’re just starting out or brushing up your skills, understanding Git’s core concepts and commands is key to becoming a confident coder. Below is a concise and practical overview of essential Git elements and commands, perfectly laid out in a simple table to help you navigate this powerful tool with ease.





Git Command Function
git --version Shows installed Git version
git init Initializes a new Git repository inside the folder
git add file.html Adds specified file to staging area
git add . Adds all changes in current directory to staging
git status Shows current status of the repository
git commit Commits staged changes to the gitHub
git commit -m "message" Commits stage changes with a message
git log Shows commit history
git reset --hard copied-version-id Resets to specified commit, discarding changes.

★ copied-version-id is shown in the git log command. Just copy and paste it.
git reset --soft copied-version-id Resets to commit but keeps changes staged
git remote add origin paste-url Adds a remote repository named origin
git remote remove origin Removes the origin 'remote'
git branch -m main Renames current branch to main
git push -u origin main Pushes local main branch and sets upstream
git clone gitrepositorylink Copies a remote repository locally
git pull Fetches and merges changes from remote

Remember, every expert started with these same building blocks, so keep experimenting, learning, and having fun with it—there’s a whole world of creativity waiting for you!
